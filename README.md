## 命令行式的解释器 —— 贤者解释器
萌新大佬开发
最初目的是弄得沙盒
然而沙盒总会被破解
所以搞了这个东西
之后将用于制作地图
非常基础
其中index.js是个例子，XZ.js是主体
这种“语言”借助javascript编写运行，而不是单独的文件

## 指令
- start: 开始程序/一个代码块
- over: 结束程序/一个代码块
- add: 加法，传入三个参数，将前两个相加后传入第三个参数所指向的存储位
- mul: 乘法，用法与加法类似
- create: 创建存储位，一个参数，即存储位名称，仅限英文字母
- write: 写入，两个参数，将参数一写入参数二所指向的存储位
